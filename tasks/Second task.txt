1) Начать реализацию CLI интерфейса пользователя.
Реализовать меню при старте приложения:
	- Просмотр всех доступных вольеров (При выборе это команды должен в столбик показываться список всех вольеров)
	- Создание нового вольера (При выборе этой команды должна запустится процедура создания нового вольера)
	- Выход (Завершение программы)
При отображении списка вольеров:
	- Должна быть возможность стрелками вверх ходить наверх списка, вниз ходить вниз списка, на клавишу enter выбрать вольер
При создании нового вольера:
	- По порядку будут выходить вопросы с какого типа вольер создать, сколько мест будет в вольере и т.д., для начальной инициализации вольера
	  пользователь будет выбирать из предложенных вариантов. Сразу после созадния вольера, он будет автоматически выбираться.
2) Продолжить реализацию CLI интерфейса пользователя.
Реализовать меню при выборе вольера:
	- Просмотр всех животных (При выборе это команды должен в столбик показываться список всех животных)
	- Заселить новое животного (При выборе этой команды должна запустится процедура заселения нового животного)
При отображении списка животных:
	- Должна быть возможность стрелками вверх ходить наверх списка, вниз ходить вниз списка, на клавишу enter выбрать животное
При заселении нового животного:
	- По порядку будут выходить вопросы какого животного создать, его начальная инициализация, после этого будет происходить выбор этого животного
	  с его описанием, характеристиками, состоянием, уровнем голода и потребностью сна.
